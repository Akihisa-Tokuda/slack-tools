1. slack appの権限の設定
https://api.slack.com/apps

User Token Scopesで以下を追加する。

admin
channels:read
files:read
files:write
users:read
users:read.email

Work Spaceにアプリを再インストールする

2. 画面上でOAuth token（xoxp始まりの文字列）をchannel.batとuser_mail.batに記載。それぞれ、２行目

3. node jsのインストール
こちらを参考にインストールしてください。
https://qiita.com/echolimitless/items/83f8658cf855de04b9ce

4. channel.batとuser_mail.batを実行いただければ、ご指定の通りslack上でファイルを削除することができます。

cmd.batのSLACK_UPLOAD_FILE_TYPEで対象のファイル種類を指定できます。
※カンマ区切りで複数種類削除できます。

[指定できる種類]
all - All files
spaces - Posts
snippets - Snippets
images - Image files
gdocs - Google docs
zips - Zip files
pdfs - PDF files

[引数での実行]
コマンドプロンプトから
channel.bat general 2
のように実行可能です。
